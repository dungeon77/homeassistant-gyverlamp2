## Gyver Lamp 2 Integration

Full-featured integration for Gyver Lamp 2 WiFi LED controllers with complete UDP protocol implementation.

### Features:
- Complete preset management (25 presets)
- Real-time UDP control
- Group support (8 groups)
- All effects and palettes
- Sound reactivity
- Matrix configuration
- Native HA UI

[View documentation on GitHub](https://github.com/dungeon77/homeassistant-gyverlamp2)
